Pre-Roll Videos Enhanced

Play pre-roll videos before movies and TV shows on ALL clients using server-side session management.

Select the library containing your pre-roll videos. Videos should be stored as "Movies" content type.

Enable Pre-Rolls For:

Play pre-rolls before movies
Play pre-rolls when starting a TV series
Play pre-rolls before each TV episode

Metadata Selection Rules

Use metadata to intelligently match pre-rolls to content. If no rules match, a random pre-roll will be selected.

Match pre-rolls with a tag matching the content name
Match pre-rolls with a tag matching the release year (e.g., "1985")
Match pre-rolls with a tag matching the decade (e.g., "1980s")
Match pre-rolls with seasonal tags currently in season (defined below)
Match pre-rolls with overlapping genres
Match pre-rolls with overlapping studios
When enabled, pre-rolls must match ALL selected tag rules. Default is ANY (OR logic).

Additional Options

Filter out pre-rolls with seasonal tags that are not currently in season
Ensure pre-roll ratings do not exceed content ratings (e.g., don't show R-rated pre-rolls before PG-13 content)
Minimum seconds between pre-roll triggers for the same content (prevents loops)

Seasonal Tag Definitions

Define time periods for seasonal pre-rolls (e.g., Christmas, Halloween). The year will be ignored - only month/day matter.
